# My Hyprland Dotfiles for Arch Linux

I don't have much time these days so it's quite messy, but its clean enough for you to clone and use immediately.

Run the `link.sh` file to symlink the dotfiles to `~/.config/`. 

This repo also houses my NixOS dotfiles, which you probably dont need. To clone only this branch, run:

```bash 
git clone https://github.com/Natzukashi/dotfiles --single-branch -b master
```

Please go through the config to understand what to do. It's pretty self-explanatory.
